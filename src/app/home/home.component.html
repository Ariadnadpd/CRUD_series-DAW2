eSeries, tu banco de series virtual

eSeries es una aplicación desarrollada con el fin de proporcionar una base de datos basada en las series favoritas de Netflix.

Además, con eSeries puedes añadir, editar o eliminar las series o películas que desees.

Inicia sesión o regístrate para poder ver el listado de series y películas.
