eSeries

Esta aplicación está desarrollada por Ariadna Del Pino Domínguez. Es una aplicación SPA que contiene una base de datos basada en algunas series de Netflix. Para poder ver el contenido de la base de datos, editarlo, crear un nuevo registro o eliminarlo, accede haciendo clic en el botón que se encuentra a continuación.


Información sobre la base de datos:

  • La base de datos de series contiene series de netflix de diferentes géneros: thriller, ciencia ficción, acción y aventuras, basadas en cómic, siniestra, emocionante, inquietante, etc.
  • Además, contiene los siguientes campos de información: año de lanzamiento, país, temporadas, episodios, géneros y una pequeña descripción.
